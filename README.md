## About

Code, scripts, and notes created while completing the **Master Network Automation with Python for Network Engineers** course on Udemy. Covers a wide range of tools and libraries used for automating network device management.

## Topics Covered

| Topic | Tools / Libraries |
|-------|------------------|
| SSH automation | Paramiko |
| Network device automation | Netmiko |
| Multi-vendor network automation | NAPALM |
| Telnet automation | Telnetlib |
| Async networking | asyncio, aiohttp, asyncssh |
| Data serialization | JSON, CSV, Pickle |
| REST APIs | requests, json |
| File & OS operations | shutil, subprocess |
